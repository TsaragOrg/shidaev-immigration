# Юридические страницы

## Что реализовано

5 юридических страниц на сайте Шидаева. Все билингвальны (RU + EN).

| Страница | Маршрут | Назначение |
|---|---|---|
| **Legal Notice** (Правовое уведомление) | `/legal` + `/en/legal` | Реклама юридических услуг, отказ от ответственности, отсутствие отношений «адвокат-клиент» |
| **Privacy Policy** (Политика конфиденциальности) | `/privacy` + `/en/privacy` | Какие данные собираем, как храним, права пользователя |
| **Terms of Use** (Условия использования) | `/terms` + `/en/terms` | Допустимое использование, IP, ограничение ответственности, юрисдикция |
| **Accessibility Statement** (Заявление о доступности) | `/accessibility` + `/en/accessibility` | Соответствие WCAG 2.1 AA, ADA Title III, альтернативные каналы связи |
| **Fraud Warning** (Предупреждение о мошенничестве) | `/fraud-warning` + `/en/fraud-warning` | Об immigration scams — как защититься |

## Юрисдикция

US (California State Bar). Тексты прошли проверку с учётом:
- California Bar Rule 1-400 (false advertising — нет утверждений о прокурорском опыте)
- Pro hac vice квалификации (с осторожной формулировкой)
- ADA Title III (a11y декларация)

## Что НЕ применяется к Шидаеву

- RGPD (EU) — клиент US
- LGPD (Brazil) — N/A
- mentions légales (FR) — N/A (есть в `~/.claude/templates/` для FR-проектов)

## Общая обвязка

- **`LegalLayout.tsx`** — общая обёртка для всех 5 страниц
- **Стили:** `41-legal.css`
- **Контент:** живёт в `src/lib/legal/` (отдельные файлы для каждого документа)
- **Footer:** 4 ссылки на все legal-страницы

## Что НЕ менять без юридической проверки

- Формулировки про admissions / pro hac vice
- Disclaimer «no attorney-client relationship»
- Перечень bar admissions с цифрами
- Compliance-формулировки в Fraud Warning
